# Color System

The UI ratio is 85% neutral, 10% blue, 5% red.

| Color | Hex | Use |
|---|---|---|
| Black | `#000000` | Main text, strong icons |
| Blue | `#0061FF` | Primary button, active tab, focus, today/action emphasis |
| White | `#FFFFFF` | Page/card/dialog background |
| Red | `#DE0000` | Missing docs, delete, errors |
| Muted | `#8B8783` | File names, metadata, secondary labels |

Event categories: Training bg `#EAF2FF` accent `#0061FF`; Match bg `#FFF4E0` accent `#B56A00`; Meeting bg `#F2EEFF` accent `#6E42D9`; Medical bg `#EAF7EF` accent `#0F8A3B`.
