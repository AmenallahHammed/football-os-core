\chapter*{General Conclusion}
\addcontentsline{toc}{chapter}{General Conclusion}

This internship project aimed to design and implement Football OS, a modular platform for football organizations that centralizes key operational workflows in one secure workspace. The initial objective was to move from scattered tools to an integrated system that supports authentication, role management, document management, calendar and event management, notifications, search, and player profile consultation.

The work progressed from requirements analysis to architecture and implementation. The requirements were formalized through use cases and diagrams, then translated into a design centered on the API Gateway, Governance Service, Workspace Service, and Shared SDK, supported by Keycloak, OPA, MongoDB, PostgreSQL, MinIO, Kafka, and OnlyOffice.

The implementation delivered a functional platform foundation. Secure access and role-based authorization were integrated through identity and policy components. Document workflows now support upload, consultation, archiving, and online editing; calendar workflows support attendees, tasks, required documents, and notifications; and player profile consultation is filtered by role permissions. Persistence is distributed between MongoDB and PostgreSQL, while file content is stored in MinIO. The Shared SDK reduced repeated technical code across backend services.

This work still has limitations. Some frontend/backend integrations require final polishing, and parts of the interface remain at prototype level, especially around full player profile API integration. Broader testing, deployment validation, and monitoring practices are also needed to improve production readiness.

Future work should focus on completing the remaining integrations, improving user experience, and strengthening automated testing at unit, integration, and end-to-end levels. It should also include stronger monitoring and deployment workflows, richer notification and search capabilities, and more advanced football-specific operational processes to consolidate Football OS as a mature production-ready platform.
